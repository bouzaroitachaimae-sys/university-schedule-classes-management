# University Classes & Schedule Management System

A web application to manage room reservations, academic schedules and automated timetable generation.

## Features
- Room reservation management
- Automated timetable generation
- Semester Schedules
- Convocation production
- Role-based access control (Admin, Teacher, Student)
- Intuitive and user-friendly interface

## Technologies Used
- Python
- Flask
- HTML
- CSS
- MVC Architecture

## Installation
1. Clone the repository
2. Install dependencies:
   pip install -r requirements.txt
3. Run the app:
   python app.py

## Roles
- **Admin** - Full access
- **Teacher** - Manage schedules
- **Student** - View timetables
